# Tag Management Rules

## Tag Categories

| Category | Format | Example |
|----------|--------|---------|
| Date | `YYYY/MM/DD` | `2024/12/13` |
| Note Type | Uppercase | `LITERATURE`, `PERMANENT`, `STRUCTURE` |
| AI Generated | Uppercase | `WRITTEN_BY_AI` |
| Domain | Discovered from existing notes | - |

## Rules

1. **Namespace Conservation**: Avoid creating unnecessary new tags
2. **Consultation Required**: Always check existing tags before suggesting new ones
3. **Consistency**: Maintain consistent tag naming conventions
